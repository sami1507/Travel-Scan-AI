# TravelScan AI

AI-powered travel analysis platform built with **Next.js**, **Node.js**, **OpenAI**, and **Supabase**.

TravelScan AI helps users make smarter travel decisions by analyzing destinations, routes, timing, budget fit, accommodation value, weather, seasonality, and travel preferences through structured, data-driven recommendations.

## Overview

This project is designed as a **travel intelligence platform**, not just a chatbot.

Instead of returning generic travel suggestions, the system combines:
- structured travel knowledge
- provider-based data
- deterministic scoring
- route intelligence
- personalization
- feedback learning
- analytics

to generate clearer, more explainable, and more useful travel recommendations.

## Core Capabilities

- Destination analysis
- Route intelligence
- Budget fit scoring
- Weather and seasonality analysis
- Passport / visa ease analysis
- Flights and hotels integration
- Personalized recommendations
- Saved trips and travel history
- Comparison mode
- Alerts and notifications
- Admin analytics dashboard
- Structured JSON AI output
- Evaluation and verification layers for recommendation quality

## Tech Stack

- **Frontend:** Next.js, React, TypeScript
- **Backend:** Node.js, Next.js API routes / server actions
- **AI Layer:** OpenAI
- **Database & Auth:** Supabase
- **Styling/UI:** Tailwind CSS + custom UI components

## Goal

The goal of this product is to move beyond basic travel search and become a **decision-support platform for travelers** — helping users understand not only *where to go*, but also *why*, *when*, and *how* to choose the best option.

## Status

Active development. The platform includes the core travel analysis engine, personalization, route intelligence, admin analytics, and user feedback systems.
